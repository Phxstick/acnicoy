Import vocabulary
Acnicoy supports importing vocabulary items from UTF-8 encoded CSV files. for information on accepted fields.
Data type:
Vocabulary Kanji Hanzi Houhou SRS
Please note that there exist some incompatibilities between Houhou SRS and Acnicoy, which can lead to data loss during the import procedure. for more information.
Language data for Japanese must be available to allow correct assignment of kanji readings importing data from Houhou SRS. You can download that data from the language settings.
Field separator:
Tab Semicolon Comma
Item separator:
Semicolon Comma Slash
SRS levels start at 0 (instead of 1)
