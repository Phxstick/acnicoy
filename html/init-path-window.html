Please choose the directory where your data will be stored in.
This will include your vocabulary, user preferences etc. and should be stored separately from the program files, e.g. in your home directory.
If you wish to use the program across multiple devices, you can choose a folder managed by a synchronization service such as Dropbox.
If you used the program previously already and want to continue with existing data, you can select the corresponding directory now. If you use the program for the first time, specify a new or empty directory.
