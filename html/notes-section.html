  • Notes are organized into groups, shown on the left. Selecting a group displays all assigned notes.
  • Right-clicking a group opens a context menu which allows you to remove it, rename it, or add a subgroup.
  • Edit a note by double clicking it or using the context menu. Leave edit mode using Escape or Ctrl+Enter.
  • Drag a note by pressing the left mouse button while holding Ctrl (can be released after it has been grabbed).
  • Dragged notes can be moved to other groups and reordered within the same group.
