• Notes are organized into groups, shown on the left.
    Selecting a group displays all notes assigned to it.
  • Right-clicking a group opens a context menu which allows you to remove the group, rename it, or add a subgroup.
  • Edit a note by double clicking it or using the context menu. Leave edit mode by pressing Escape or Ctrl+Enter.
  • Drag a note by pressing the left mouse button while holding Ctrl. Ctrl can be released again after the note has been grabbed.
  • Dragged notes can be moved to other groups and reordered within the same group.
